Dropcraft is a NuGet-based application deployment and composition framework.
Get Started Download Latest Dropcraft

Power of NuGet in your application.

Install and update NuGet packages. Use NuGet-based plugins. Use an extensibility point concept to compose applications from the packages. All these scenarios are possible with Dropcraft without learning NuGet API.

Any platform. Any application model.

Targeting .NET Standard, Dropcraft is capable to support .NET Core and .NET Framework. It can be used with the console, desktop and Web applications.

Extensible for your needs.

Control packages deployment. Override default runtime behavior. Create additional configuration mechanisms. Easily create your own command line tools or integrate Dropcraft in your application.
